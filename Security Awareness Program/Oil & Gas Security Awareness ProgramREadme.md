# Oil & Gas Security Awareness Program

## Objective
Build role-based cybersecurity awareness for oil and gas employees, contractors, and vendors.

## Audience Tracks
| Audience | Focus |
|---|---|
| Executives | Cyber risk oversight, incident decision-making, regulatory exposure |
| Field Operators | Phishing, removable media, physical security, reporting suspicious activity |
| Engineers | Secure remote access, OT change control, data integrity |
| Corporate Staff | Email security, data handling, AI usage rules |
| Contractors/Vendors | Access rules, site expectations, incident reporting |

## Annual Program Calendar
- Q1: Phishing and credential protection
- Q2: OT/ICS security and remote access
- Q3: AI and data protection
- Q4: Incident reporting and tabletop exercise

## Metrics
- Completion rate
- Phishing simulation click rate
- Reporting rate
- High-risk user follow-up completion
- Contractor onboarding completion
